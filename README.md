## 📌 Sobre o Projeto

Este projeto é uma calculadora simples de consumo de energia elétrica.

O usuário informa:
- Nome do aparelho
- Potência (em watts)
- Tempo de uso diário (em horas)

E o sistema calcula o consumo mensal e o custo estimado.

---

## ⚙️ Tecnologias utilizadas

- 🐍 Python
- 💻 VS Code
- 🌐 GitHub

---

## 🧮 Fórmula utilizada

Consumo Mensal (kWh) = (Potência × Horas por dia × 30) / 1000

---

## ▶️ Como executar

1. Clone o repositório:
git clone https://github.com/bruno-ryu/consumo-energia.git

2. Entre na pasta:
cd consumo-energia

3. Execute:
python app.py

---

## 📊 Exemplo

Entrada:
Digite o nome do aparelho: Geladeira  
Digite a potência (em watts): 150  
Digite as horas de uso por dia: 10  

Saída:
Aparelho: Geladeira  
Consumo estimado: 45 kWh/mês  
Custo estimado: R$ 33.75
